# 시니어와 가족을 위한 시니어 말동무 & 대화 분석 리포트 제공 서비스

- 어르신의 대화 단절로 인한 외로움과 어르신과 가족간의 의사소통 부족 문제로 인한 관계 단절을 문제로 인식하였습니다.
- 어르신의 '일상 대화' 그리고 '가족간의 의사소통' 증진을 위해 **말동무 챗봇 & 대화 분석 리포트** 를 주 기능으로 잡고 프로젝트를 진행하였습니다.
